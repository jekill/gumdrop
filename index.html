Gumdrop

Static website generator converting Markdown files and using the Twig templating engine.
Quick start
php -r "$(curl -s http://gumdropapp.com/installer.php|tail -n +2)"

Content agnostic

You can build whatever you want: a blog, a technical documentation, a portfolio, ...

Built-in HTTP server

Gumdrop comes with a built-in server so you don't need to install one. For development purposes only.

Industry standards

Gumdrop uses the latest features from PHP 5.4 and is based on Composer, the best dependency manager for PHP.

Simple

Gumdrop converts Markdown and handles Twig templates.

Auto-reload

Gumdrop can watch for changes in your source file and re-render your site automatically.


Open-source

Gumdrop is open-source, free of charge, under an MIT license.
